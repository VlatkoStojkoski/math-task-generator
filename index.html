Addition Subtraction Multiplication Division COVID-19

About This Project

This project was made regarding the COVID-19 situation.

Young kids can solve randomly generated tasks without bothering their parents to write tasks for them, so that parents can focus more on their work from home.

This website motivates kids to solve more tasks by higlighting the correct results in green.
